# Desktop Control Mockups

These are illustrative dark-theme renders of the implemented Plasma QML. They
use representative healthy values from `MockBackend.qml`; they are not runtime
screenshots from BC-250 hardware.

- `01-overview-tray.svg`: compact system-tray popup and panel icon
- `02-gpu-controls.svg`: wide `plasmawindowed` GPU controls
- `03-cu-cpu.svg`: CU routing and CPU tuning views
- `04-cec-controls.svg`: CEC status, actions, and behavior toggles

PNG copies are generated locally for convenient viewing.
