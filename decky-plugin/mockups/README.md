# Decky Plugin Mockups

These are illustrative renders of the implemented Decky Quick Access and
fullscreen React interfaces. They use representative healthy values matching
the backend schema and are not runtime captures from BC-250 hardware.

- `01-quick-access.svg`: Quick Access Summary panel and telemetry
- `02-full-overview.svg`: fullscreen Overview and vertical navigation
- `03-cu-gpu.svg`: CU routing and GPU tuning tabs
- `04-cpu-cec.svg`: CPU overclock and HDMI-CEC tabs

PNG copies are included for README rendering.
